# 公開資料來源

依使用價值排序。每一項標明：URL、公開到什麼程度、姓名是否可見、使用時機。

---

## 一級來源（建議優先）

### 1. 司法院判決書 Open Data（首選）

- URL：<https://opendata.judicial.gov.tw/>
- 公開內容：所有已公布裁判書的每日批次 JSON，依法院、依日期打包成 ZIP。
- 使用方式：下載日期區間 ZIP → 在本機用 `grep` 或一段 Python 過濾姓名、法條、關鍵字 → 把命中的判決檔餵給 LLM。
- 為什麼首選：不會被搜尋頁限流、不會造成網站負擔、批次撈取效率高。
- 限制：發布有約 1–2 週延遲。
- 姓名能見度：刑事判決的**成年被告全名**通常保留；被害人、證人、少年皆遮為 ○○○。

### 2. 司法院裁判書查詢

- URL：<https://judgment.judicial.gov.tw/>
- 公開內容：與 Open Data 相同，但是互動式搜尋介面。
- 使用方式：只在你已知特定案號、想立刻看單一判決原文時使用。
- ⚠️ **不要寫腳本批次抓搜尋頁**，會被擋而且對網站造成負擔。要批次請回到 Open Data。

## 二級來源（針對特定族群）

### 3. 內政部警政署 通緝要犯查詢

- 公開內容：姓名、照片、通緝原因、通緝法院。
- 姓名能見度：全名公開。
- ⚠️ **照片著作權屬警政署**，轉貼時務必標註「照片來源：內政部警政署」。
- 用於：通緝中的當事人，特別是重大刑案逃犯。

### 4. 性侵害加害人登記公告查詢

- 範圍：僅符合性侵害犯罪防治法 §23 條件的加害人。
- 公開內容：姓名 + 戶籍直轄市/縣市區。
- 限制：範圍非常窄，多數性侵罪不在此名單。

### 5. 懲戒法院判決查詢（前 公務員懲戒委員會）

- URL：<https://www.tcc.gov.tw/>
- 對象：公務員懲戒案件。
- 姓名能見度：公務員全名保留。
- 用於：公務員違失、貪瀆相關紀錄。

### 6. 監察院糾彈案查詢

- URL：<https://www.cy.gov.tw/>
- 對象：被糾舉、彈劾的公務員與民代。
- 姓名能見度：保留。
- 用於：補強政務官、民代相關公開違失資料。

### 7. 法務部廉政署「廉政專刊」

- URL：<https://www.aac.moj.gov.tw/>
- 公開內容：歷年定期發行 PDF，列載被起訴或判決的公務員案例。
- 用於：公部門貪瀆案件清單，配合裁判書交叉確認。

### 8. 金管會、公平會、其他主管機關裁罰公告

- 例：金管會「裁罰案件查詢」、公平會「裁處案件公告」。
- 法人通常具名；個人多數匿名，少數情況具名。
- 用於：經濟犯罪、詐欺、內線交易、聯合行為。

### 9. 政府資料開放平臺

- URL：<https://data.gov.tw/>
- 用法：中央目錄，可從這裡再導向以上各來源的開放資料集。

## 不能當來源

- ❌ **法律扶助基金會檔案**：保密，禁止使用。
- ❌ **戶政紀錄、警察前科紀錄**：不公開，禁止使用。
- ❌ **兒少性剝削加害人登記**：依兒少性剝削防制條例 §31，僅通報學校、僱主等特定對象，**不對外公開查詢**，禁止散布。
- ❌ **新聞報導作為最終來源**：可以作為「線索」幫你找到案號，但最終紀錄必須引用 judgment.judicial.gov.tw 的裁判書本身。若找不到對應裁判書，整筆放棄。

## 批次撈取 vs 搜尋頁 — 心法

| 情境 | 用哪個 |
|---|---|
| 想找某段時間所有「兒少性剝削防制條例 §36」的案件 | Open Data ZIP + 本機 grep |
| 想找某姓名所有有罪確定判決 | Open Data ZIP + 本機 grep |
| 已經有案號，想立刻看原文 | judgment.judicial.gov.tw 搜尋頁 |
| 想驗證 LLM 摘要是否與原文一致 | judgment.judicial.gov.tw 直接看原文 |
| 想撈通緝犯名單 | 警政署通緝要犯查詢 |

預設答案是 **Open Data**。
