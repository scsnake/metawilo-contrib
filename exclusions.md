# 排除清單

**在進行任何 LLM 操作之前**檢查這份清單。命中任一條的案件，請不要蒐集、不要餵 LLM、不要提交。

---

## 1. 少年事件

- 少年法庭、少年保護事件、少年刑事案件。
- 被告為 18 歲以下時點犯罪、案號帶「少」字。
- 全程匿名，沒有可公開的個人資訊。

## 2. 無罪 / 不起訴 / 緩起訴

- 判決主文「無罪」、「免訴」、「不受理」。
- 檢察官「不起訴處分」、「緩起訴處分」。
- 理由：**刑法 §310 妨害名譽風險**。已經被判無罪的人若被列入「犯罪資料」彙整，可能構成妨害名譽。

## 3. 未確定判決

- 還在上訴期間 / 已上訴 / 發回更審。
- 主文可能被推翻。**等確定之後才提交。**

## 4. 性侵害案件中可能反推被害人身份

- 即使被告姓名公開，若摘要寫出「被告為被害人之繼父」、「住在某偏鄉小村」、「事發於 2024 年 3 月某日下午」等組合，外人就能反推被害人是誰。
- 性侵害犯罪防治法 §13 禁止揭露足以辨識被害人身份的資訊。
- **寧可整筆放棄，也不要冒險。**

## 5. 摘要會反推任何匿名當事人

- 不限性侵害。任何案件中，如果重點摘要透露的「關係 + 地點 + 時間」組合足以指認被遮蔽的當事人（被害人、證人、被告之家屬），整筆放棄。

## 6. 來源僅為新聞報導

- 新聞可以幫你**找到案號**，但最終紀錄必須引用 judgment.judicial.gov.tw 的裁判書本身。
- 找不到對應裁判書 → 整筆放棄。

## 7. 來自非公開或封閉系統的資料

- 法律扶助基金會檔案。
- 戶政、警察前科紀錄。
- 兒少性剝削加害人登記名單。
- 即使有人提供給你，也不能蒐集進這個專案。

## 8. 站長標示為敏感的案件

- 若站長在 README 或 GitHub Issues 加列額外排除規則，以站長為準。

---

## 倫理警示（不是硬排除，但請先請示站長）

- **超過 10 年且已執行完畢的案件**：被告可能已是更生人，重新被當作「犯罪資料」公開可能阻礙重新融入社會。提交前請在 PR 描述標註「更生人疑慮」，由站長判斷。
- **罰金易科或微罪**：例如易科罰金、酒駕初犯、輕微傷害。集中蒐集這類紀錄與本網站「警示重大公益相關犯罪」的宗旨可能不符。
- **被告為政治人物或宗教領袖**：政治敏感性高，建議站長個案審核而非開放貢獻。

---

## 不確定怎麼辦？

當你不確定一筆案件能不能蒐集，**預設答案是「不要蒐集」**。可以在 GitHub Issues（或站長指定通道）發問。
